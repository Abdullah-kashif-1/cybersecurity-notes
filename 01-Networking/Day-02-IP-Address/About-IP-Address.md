# 🌐 IP Addressing Basics

## 📌 What is an IP Address?
An IP (Internet Protocol) Address is a unique identifier assigned to each device connected to a network. It allows devices to communicate with each other over the internet or local networks.

---

## 🔑 Key Points:
- Every device has a unique IP address  
- It works like a home address for devices  
- Helps in sending and receiving data  
- Essential for internet communication  

---

## 🌍 Types of IP Addresses:

### 1. IPv4
- Most commonly used  
- Written in numbers (e.g., 192.168.1.1)  
- Limited number of addresses  

### 2. IPv6
- New version of IP addressing  
- Uses hexadecimal format  
- Provides a large number of addresses  

---

## 🔐 Public vs Private IP:

### Public IP
- Assigned by Internet Service Provider (ISP)  
- Used on the internet  
- Accessible globally  

### Private IP
- Used within local networks  
- Not accessible from outside  
- Common in home/office networks  

---

## ⚙️ Static vs Dynamic IP:

### Static IP
- Fixed IP address  
- Does not change over time  
- Manually assigned  
- Used for servers, websites, and important devices  

### Dynamic IP
- Automatically assigned by DHCP  
- Changes over time  
- Commonly used in home networks  
- More secure and cost-effective  

---

## 💡 Example:
When you connect your phone to Wi-Fi, it gets a private and usually dynamic IP, while your router has a public IP on the internet.

---

## 🎯 Conclusion:
IP addressing is a fundamental concept in networking that enables communication between devices across networks.
