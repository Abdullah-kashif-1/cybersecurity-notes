## 💻 IP Address Related CMD Commands

### 1. ipconfig
**Use:**  
- Shows your system's IP address  
- Displays IPv4, IPv6, subnet mask, and default gateway  

---

### 2. ipconfig /all
**Use:**  
- Shows detailed network information  
- Includes MAC address, DHCP status, DNS, etc.  

---

### 3. ipconfig /release
**Use:**  
- Releases current IP address  
- Disconnects from network temporarily  

---

### 4. ipconfig /renew
**Use:**  
- Requests a new IP address from DHCP server  
- Reconnects to network  

---

### 5. ipconfig /flushdns
**Use:**  
- Clears DNS cache  
- Helps fix website loading issues  

---

### 6. ping [IP/Website]
**Example:** ping google.com  
**Use:**  
- Checks connectivity between your system and another device  
- Measures response time  

---

### 7. tracert [IP/Website]
**Example:** tracert google.com  
**Use:**  
- Shows the path data takes to reach destination  
- Helps identify network delays  

---

### 8. nslookup
**Use:**  
- Finds IP address of a domain  
- Useful for DNS troubleshooting  

---

### 9. netstat
**Use:**  
- Displays active network connections  
- Shows open ports and listening services  

---

### 10. arp -a
**Use:**  
- Displays IP to MAC address mapping  
- Useful for local network analysis  

---
